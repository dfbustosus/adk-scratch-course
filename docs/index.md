## ğŸš€ Quick Start

Get up and running in minutes:

```bash
# Install the framework
pip install adk-course

# Validate your environment
adk-course validate

# Create your first agent
adk-course init my-agent

# Test your agent
adk-course test --config agents/my-agent/config.yaml
```
